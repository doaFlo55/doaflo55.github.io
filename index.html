Audio Visualizer Protocol

To implement the visualizer, use npm i audiomotion-analyzer. Ensure your audio element has the crossorigin property set to "anonymous".

// Initialize
const audioMotion = new AudioMotionAnalyzer(container, { source: audioSource });

// Combining Multiple Sources
audioMotion.connectInput(audioElements[0]);
audioMotion.connectInput(audioElements[1]);
